# plug_control
基于机智云和 esp8266-12F WIFI 模块的智能插座控制安卓APP

##  feature

- [x] 手机插座WiFi配对
- [x] 远程控制开关
- [x] 定时开启/关闭
- [x] 倒计时开启/关闭
- [x] 温度/湿度显示
